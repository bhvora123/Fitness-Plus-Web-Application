Adding functionality for adding exercises to training days

About the experiment

In this experiment I have created the basic layout for adding exercises. Here in input for exercises user can enter a excerise name. Here I created a auto complete using JQuery where what ever user write will be evaluated by my auto complete and it will find the relevant exercies from the service and display in UI along with categories.

All form fields are required.

You can search for one exercise, they will be grouped together for a superset. Sample "bicep"

Enter the number of repetitions for all sets, you can just enter one value: e.g. for 4 sets just enter one "10" for the repetitions, this automatically becomes "4 x 10".
